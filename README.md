Indoor Plant Growth Controller

This project is an indoor cultivation study system built with Arduino to automate environmental controls for plant growth. The system is designed to manage lighting cycles, temperature, air humidity, and soil moisture using modular controllers and sensor-based logic.

The software architecture is organized into independent domains, making the project easy to maintain, expand, and test. The goal is to provide a simple and scalable platform for learning embedded systems, automation, and environmental monitoring in indoor agriculture.
